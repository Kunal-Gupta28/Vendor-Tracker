# Vendor-Tracker

A modern Vendor Tracking Web Application built with Next.js, TypeScript, Tailwind CSS, and Node.js.  
This project helps manage vendors, track activities, and maintain organized vendor-related workflows.

## Live Demo

Deployed application:  
https://vendor-tracker.vercel.app/

## Features

- Vendor Management  
  Add, view, update, and delete vendors.
- Clean and User-Friendly Interface  
  Simple and intuitive UI for easy navigation.
- Responsive Design  
  Optimized for desktop and mobile devices.
- Scalable Codebase  
  Modular structure that allows easy future enhancements.

## Tech Stack

| Technology | Purpose |
|------------|---------|
| Next.js | React framework with server-side rendering |
| TypeScript | Type-safe JavaScript |
| Tailwind CSS | Utility-first CSS framework |
| Node.js | JavaScript runtime |
| Vercel | Deployment platform |
